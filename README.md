# Colorado Legal Service

A premium React + Express landing page for Colorado self-representation legal support, with intake collection and secure payment checkout.

## Features
- Premium landing page for Colorado legal filing guidance
- Intake form for case review requests
- Secure $100 payment onboarding flow
- Responsive design for desktop and mobile

## Local Development
```bash
npm install --prefix client
npm install --prefix server
npm run dev
```

## Production Build
```bash
npm run build
```

## Deployment Notes
- Frontend: Vite + React
- Backend: Express API
- Payment: Stripe Checkout
- Email notifications: SMTP via Nodemailer
